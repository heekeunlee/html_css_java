웹 앱을 만드는 마법의 레시피

HTML, CSS, 그리고 JAVA(JavaScript)가 어떻게 조화를 이룰까요?

🏗️

HTML

"집의 뼈대와 벽"

웹사이트의 구조를 만듭니다. 제목, 본문, 이미지, 버튼 등 무엇이 어디에 있을지 결정하는 설계도와 같습니다.

  • 제목 (h1, h2)
  • 단락 (p)
  • 목록 (ul, li)
  • 이미지 (img)
🎨

CSS

"인테리어와 페인트"

웹사이트의 디자인을 담당합니다. 색상, 글꼴, 간격, 배치 등을 아름답게 꾸며주는 드레스업 과정입니다.

  • 배경 색상
  • 폰트 스타일
  • 레이아웃 (Flex, Grid)
  • 애니메이션
⚡

JAVA / JS

"전기 장치와 가전제품"

웹사이트의 동작과 로직을 담당합니다. 버튼을 눌렀을 때의 반응, 데이터 처리 등 살아있는 기능을 부여합니다.

  • 클릭 이벤트
  • 데이터 계산
  • 서버 통신
  • 복잡한 로직
